Intro and The World Wide Web

The web enables clients (computers and browsers) to exchange information (text, images, audio, etc.) via the internet with servers (basically computers designed to store and send data).

These components communicate with each other through a protocol called Hypertext Transfer Protocol (HTTP) and present information in Hypertext Markup Language (HTML),which is the language browsers use to interpret webpages.

The Basics of HTML

When coding in HTML or any other language, we are in essence writing instructions for a computer to perform an action. The inability (so far!) of computers to successfully deduct what we want from context, requires us to be precise, otherwise the computer will not understand, which in turn leads to unnecessary frustration. Hence the need for specifc instructions.

Components of HTML

HTML intructions include:

(1) Elements- what we use to tell the computer how to display a page. Usually composed of content, tags and in some cases attributes.

(2) Content- that which is displayed (text, images, etc.)

(3) Tags- indicates how the content is displayed and in most cases requires an opening and closing tag. Take as an example the tag: "<b>content</b>" which makes the word "content" bold. Another common tag is: <em>content</em>, which renders the text in italics.

(4) Attributes- some tags allow for additional "instructions", which follow the format name="value". For example the anchor tag: "<a href="URL"&g;content</a>", tells the browser to create a link (from the word "content") to a specific URL. (Note: values seems to be always within "").

Some elements are considered block elements, which are elements that create "blocks" or sections in the code or the webpage. In contrast, inline elments do not create separate sections.

Examples of block elments include: <p>content</p>, and <div>content</div>, which indicate the creation of a new paragraph when writing text on a webpage and the definition of a separate "box" section or container in an HTML document respectively.

Examples of inline elements include: <br> and <span>content</span>. The former indicates a line break in the text and also happens to be a void tag (i.e. it doesn't need a closing).The latter creates an inline container element (as opposed to <div>content</div>).
